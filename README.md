# Ocean View Resort Online Reservation System

## Overview
The Ocean View Resort Reservation System is a Spring Boot based web application developed to manage hotel room reservations efficiently. The system allows administrators and users to manage room bookings, guests, billing, and reports.

## Technologies Used
- Java
- Spring Boot
- Maven
- MySQL
- REST API
- JWT Authentication
- HTML/CSS

## System Features
- User authentication and authorization
- Room management
- Guest management
- Reservation booking system
- Billing management
- Report generation

## System Architecture
The system follows a layered architecture:
- Controller Layer
- Service Layer
- Repository Layer
- Entity Model

## Project Structure
src/main/java/com.oceanview.resort
├── controller
├── service
├── repository
├── model
├── config

 How to Run the Project

1. Clone the repository
